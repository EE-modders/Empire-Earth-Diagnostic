# Empire-Earth-Diagnostic
A simple tool to get the installation information of the community setup.

✅ Windows Version
✅ Setup Version
✅ GPU Name
✅ Selected Tasks
✅ Selected Components
✅ Install Mode (Regedit)
✅ Setup/Game update checker
✅ EE & AoC CRC checksum
✅ List number of custom files
